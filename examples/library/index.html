Meiosis Example - Library

To run this example:
  • Go to the meiosis-examples/examples/library directory
  • Run these commands: 
    npm i
npm start
npm run server
  • open http://localhost:8000
  • If there is a conflict with port 8000, you can specify a port (4321 in this example, but you can use a different number): 
    PORT=4321 npm run server
